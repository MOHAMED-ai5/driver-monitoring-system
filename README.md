# 🚗 Driver Monitoring System (AI Powered)

A real-time AI system built using Deep Learning and Computer Vision to detect driver drowsiness, distraction, and overall driving state in order to improve road safety.

##  Problem Statement
Driver fatigue and distraction are among the leading causes of road accidents worldwide. This project aims to detect unsafe driving behavior in real-time using AI.
